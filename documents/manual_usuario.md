# Manual de Usuario – HelpDesk

## Introducción
El sistema HelpDesk permite a los usuarios crear y gestionar tickets de
soporte técnico de manera sencilla.

## Inicio de Sesión
1. Acceder a la página de inicio.
2. Ingresar nombre de usuario y contraseña.
3. Presionar el botón de inicio de sesión.

## Crear un Ticket
1. Presionar la opción "Create Ticket".
2. Ingresar el título y la descripción del problema.
3. Enviar el ticket.

## Ver Tickets
- El usuario puede ver una lista de sus tickets.
- Cada ticket muestra su estado y prioridad.

## Comentarios
- Dentro de un ticket, el usuario puede agregar comentarios.
- Los comentarios permiten la comunicación entre usuarios y agentes.

## Cerrar Sesión
- Presionar el botón "Logout" en la barra de navegación.

*El administrador puede visualizar la lista de usuarios registrados.
